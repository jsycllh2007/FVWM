配置文件是从 [box-look](http://box-look.org/content/show.php/Lethe?content=91022) 上下载下来的，然后修修改改成了个雏形，还有许多功能没有实现，但是作为日常用的窗口管理器也差不多了。

####平台
- FreeBSD 10.2-RELEASE

####需要用到的软件
- compton
窗口阴影效果
- ImageMageick
窗口最小化缩略图显示
- xdm
同上
- feh
壁纸
- gtkchtheme
主题修改
- fcitx
默认用的输入法

####用到的主题
- gtk-murrina-fancy-clearlooks

####用到的图标
- gnome-icons-faenza

以及显示时间的脚本都已经打包在文件里面。
